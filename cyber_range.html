Level 2: Attack Chain Builder

PractiseLocked

Drag the events into a sensible order. This is about recognising a chain of risk, not learning how to attack.

  1. User receives a message designed to create urgency
  2. User clicks a link or opens an unexpected attachment
  3. Malware is installed or credentials are stolen
  4. Files or accounts are accessed without permission
  5. Defenders isolate the device, report the incident and restore safely
